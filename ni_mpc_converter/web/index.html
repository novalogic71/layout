Quick Guide

Use this checklist before running a batch.

1. Pick source folders NI roots are exact Maschine expansion folders. NI parents scan direct child folders for expansions. Sample roots are WAV pack folders. Sample parents scan direct child folders that contain WAV files.
2. Choose the kit recipe Per folder creates one kit per immediate sample-pack folder. Single mixed kit combines categories into one kit. Darkchild and Custom anchor roles to selected pads before filling the rest.
3. Set output and template Output root is where new `.xtd` kits are written. MPC root points to your MPC library so the app can find a template `.xtd`. Existing kits are skipped, not overwritten.
4. Local desktop builds Push this repo to GitHub. The Build Desktop Apps action creates macOS and Windows local launchers that start the web app and open it in the browser.
